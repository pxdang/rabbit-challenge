 # 第一章：線形代数

 #### スカラー　
大きさを表す<br/>
例：1,2,3

 #### ベクトル
向きを表す<br/>
例:
 ```
[1, 2]

[1]
[2]
 ```
 #### 行列
スカラーを表にしたもの<br/>
ベクトルを並べたもの<br/>
例:<br/>
 ```
 [1, 2]
 [3, 4]
 
 [1, 2, 3]
 [4, 5, 6]
 ```

 #### 行列の積
 ```
 [a, b]   [e]   [ae + bf]
 [c, d] . [f] = [ce + df]
 
 [a, b]   [e, f]   [ae + bg, af + bk]
 [c, d] . [g, k] = [ce + dg, cf + dk]
 ```
連立方程式の研究の中から生まれたらしい

  #### 連立方程式
 ```
 {    X1 +  4 * X2 = 7
 {2 * X1 +  6 * X2 = 10
 ```
 行列の積にすると
  ```
 [1, 4]   [X1]   [7]
 [2, 6] . [X2] = [10]
 ```
 #### 単位行列
```

```
