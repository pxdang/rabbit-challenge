 # 第一章：線形代数

 #### スカラー　
大きさを表す<br/>
例：1,2,3

 #### ベクトル
向きを表す<br/>
例:[1,2]

 #### 行列
スカラーを表にしたもの<br/>
ベクトルを並べたもの<br/>
例:<br/>
[[1,2,3],<br/>
[4,5,6]]<br/>


 #### 行列の積
 ```
 [a, b]   [e]   [ae + bf]
 [c, d] . [f] = [ce + df]
 
 [a, b]   [e, f]   [ae + bg, af + bk]
 [c, d] . [g, k] = [ce + dg, cf + dk]
 ```
連立方程式の研究の中から生まれたらしい
 
 #### 単位行列
```
[1, 0]
[0, 1]

[1, 0, 0]
[0, 1, 0]
[0, 0, 1]
```
 #### 逆行列
連立方程式を解くため
 
 #### 行列式
逆行列が存在するかどうかの条件<br/>
0の場合、逆行列が存在しない。

 #### 行列式の求め方
```
|a b|
```
