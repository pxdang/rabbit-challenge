## 正方形配列
 #### 固有値・固有ベクトル
 ```
 [1, 4]   [1]       [1]
 [2, 3] . [1] = 5 . [1]
 ```
 固有値：5<br/>
 固有ベクトル：
 ```
 [1]
 [1]
 ```
 #### 固有値・固有ベクトルの求め方
 A.X = lambda.X<br/>
 (A - lambda.I).X = 0 I:単位行列<br/>
 →|A - lambda.I| = 0 を解く<br/>
 例
 ```
 
 A=[1, 4]
   [2, 3]
 | 1 - lambda, 4          |
 | 2         , 3 - lambda | = 0
 ->(1 - lambda)(3 - lambda) - 8 = 0
 -> lambda = 5 or lambda = -1
 
 ○lambda = 5の場合
 [1, 4] [x1]     [x1]
 [2, 3].[x2] = 5.[x2]
 -> x1 + 4.x2 = 5.x1
    -> x2 = x1
 -> 2.x1 + 3.x2 = 5.x2
   -> x1 = x2
  =>固有ベクトル： [1]
                 [1]
 ○lambda = −1の場合
 [1, 4] [x1]      [x1]
 [2, 3].[x2] = −1.[x2]
 -> x1 + 4.x2 = -x1
   -> x2 = -1/2 . x1
 -> 2.x1 + 3.x2 = -x2
   -> x2 = -1/2. x1
 =>固有ベクトル：[1]
               [-1/2]
 ```
 #### 固有値分解
 ```
 A = [a, b, c]
     [d, e, f]
     [g, h, j]
 lambda1,2,3:固有値
 v1,2,3:固有ベクトル
 
 L = [lambda1, 0       , 0      ]
     [0      , lambda2 , 0      ]
     [0      , 0       , lambda3]
 
 V = [v1, v2, v3]
 
 A = V.L.V-1
 V-1:Vの逆行列
 
 ```
 配列をn乗計算する際、<br/>
 固有値分解により、固有値の対角配列をn乗計算になるため、<br/>
 簡単に計算できる。<br/>

  ## 確率
  #### 頻度確率（客観確率）
  発生する頻度。<br/>
  全集合を調査できて、確率を決める。
  
  #### 頻度確率（主観確率）
  信念の度合い<br/>
  全集合を調査できないが、様々な条件を使って、主観的に、確率を観測する。
  
  #### 条件付き確率
  P(Y|X):事象Xが与えられた下で、事象Yとなる確率<br/>
  P(Y,X):事象Yと事象Xが同時になる確率<br/>
  ```
  P(Y|X) = P(Y,X) / P(X)
  ```
  #### 独立な事象の同時確率
  独立な事象とは、因果関係のない事象とのこと
  ```
  P(X,Y) = P(X).P(Y) = P(Y,X)
  ```
  #### ベイズ則
  ```
  P(X|Y).P(Y) = P(Y|X).P(X) 
  ```
  #### 条件付き確率と同時確率の考察
  条件付き確率:P(X|Y)<br/>
  Yに対する、XとYの共通部分の割合<br/>
  
  同時確率:P(X,Y)<br/>
  Uに対する、XとYの共通部分の割合<br/>
  <br/>
  よって、同時確率 ＜ 条件付き確率
  
   #### 集合と確率
  ```
  P(X ∪ Y) = P(A)+P(B) - P(A ∩ B)
  
  AとBが独立の場合
  P(X ∪ Y) = P(A)+P(B)
  ```

## 期待値
𝑃(X):確率<br/>
f(X):確率変数<br/>
離散値の場合
```
E(f) = Σ𝑃(X).f(X)
```
連続値の場合
```
E(f) = ∫𝑃(X).f(X)dx
```

## 分散
各値は期待値からどのぐらい、ズラたか<br/>
という散らばり具合
```
Var(f) = E( (f(X) - E(f))^^2 )
       = E(f^^2) - E(f)^^2
```

## 共分散
二つのデータの傾向の違い<br/>
正の値:似た傾向<br/>
負の値:逆の傾向<br/>
ゼロ:関係性が低い
```
Conv(f,g) = E( (f(X) - E(f)) . (g(X) - E(g)) )
          = E(f.g) - E(f)E(g)
```
